# 竞彩足球比赛数据获取与盘口分析系统

实时获取竞彩足球比赛数据、赔率指数和分析数据的工具，支持大屏图形界面展示。

## 功能特性

- 实时获取比赛场次信息
- 提取赔率指数（欧洲指数、亚洲盘口、进球数）
- 获取竞彩指数数据（胜平负、比分、半全场）
- 即时走势对比（多公司赔率比较）
- 球队积分排名（总/主/客）
- 对赛往绩和近期战绩
- 半全场/进球数统计
- 大屏图形界面展示
- 联赛和状态筛选
- 自动刷新机制（1-3 分钟随机间隔）
- 数据缓存（提升加载速度）

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python gui_main.py
```

**界面功能：**
- 左侧显示比赛列表，右侧显示比赛详情
- 点击比赛卡片查看详细赔率数据
- 支持按联赛和比赛状态筛选
- 点击刷新按钮手动刷新数据
- 自动刷新数据（1-3 分钟随机间隔）
- 日期选择器切换不同日期的比赛

**盘口数据显示：**
- 比赛信息（对阵、时间、盘口）
- 即时走势（让球/大小球/欧洲指数对比）
- 竞彩指数（胜平负、亚让、进球数、比分、半全场）
- 公司赔率（欧洲指数/亚洲盘口/进球数）
- 积分排名（总/主/客积分榜）
- 对赛往绩和近期战绩
- 半全场/进球数统计

## 数据源

- 比赛列表：jc.titan007.com
- 赔率分析：zq.titan007.com

## 注意事项

1. 需要稳定的网络连接
2. 数据仅供参考
3. 建议合理设置刷新间隔

## 技术栈

- Python 3.7+
- tkinter: GUI 界面
- requests: HTTP 请求
- beautifulsoup4: HTML 解析
- lxml: XML/HTML 解析器
